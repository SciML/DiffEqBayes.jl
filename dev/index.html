DiffEqBayes.jl: Bayesian Parameter Estimation for Differential Equations

This repository is a set of extension functionality for estimating the parameters of differential equations using Bayesian methods. It allows the choice of using StanSample.jl, Turing.jl, DynamicHMC.jl and ApproxBayes.jl to perform a Bayesian estimation of a differential equation problem specified via the DifferentialEquations.jl interface.

Installation

To install DiffEqBayes.jl, use the Julia package manager:

using Pkg
Pkg.add("DiffEqBayes")
